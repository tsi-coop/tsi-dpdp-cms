System Integration

Unified API specifications for Data Fiduciaries and Processors to manage the end-to-end lifecycle of a Data Principal.

Global Request Header Requirements

# Common Auth & Content-Type
Content-Type: application/json; charset=UTF-8
X-API-Key: <UUID_CLIENT_KEY>
X-API-Secret: <HMAC_SHA256_SECRET>
